## 📞 Telephone Number Validator (HTML, CSS & JavaScript Project)

### 📌 Preview  
An interactive web app that allows users to enter a value and instantly check whether it is a valid United States phone number, while keeping a history of results.

---

### 📖 About the Project  
This project focuses on input validation and pattern matching using JavaScript. I built a Telephone Number Validator that checks whether a user’s input matches accepted U.S. phone number formats.

The application supports multiple valid formats (with or without country code, spaces, dashes, or parentheses) and provides immediate feedback by displaying results in a running history list. Users can also clear the history for a fresh start.

---

### 🚀 What I Learned  
- Working with **regular expressions (RegEx)** to detect patterns in user input  
- Implementing **custom validation logic** for multiple accepted formats  
- Manipulating strings using methods like `.replace()` and `.match()`  
- Dynamically creating and appending elements to the DOM  
- Handling **user interactions** with multiple buttons (check & clear)  
- Managing and updating a **history log** of results  

---

### 🛠️ Technologies Used  
- HTML5  
- CSS3  
- JavaScript  

---

### 🎯 Key Takeaway  
This project strengthened my ability to validate complex user input and work with pattern recognition in JavaScript. It also helped me understand how to build more interactive applications by combining logic, DOM manipulation, and user feedback.
